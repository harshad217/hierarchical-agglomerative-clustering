cho.txt and iyer.txt are the two datasets

*********************
** Data set format **
*********************

Each row represents a gene:
1) the first column is _id.
2) the second column is the ground truth clusters. The ground truths should only be used for evaluation (i.e., compute external index). "-1" means outliers.
3) the remaining columns represent attributes.
